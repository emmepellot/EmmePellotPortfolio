Projects & Graphics

Plannify

This is a modern planner app concept that was created in Figma. My team and I had an idea to create a healthy lifestyle planner app to keep track of your daily tasks, set goals, and improve organization skills!

"The Daily Mail" Website Redesign

I created a new mockup of The Daily Mail website in Figma with the inclusion of an accessibility toolbar, and overall less information overload. The purpose of this project was to make changes to a website that is not accessible to all.

Exercise App

This is an exercise app I created using React Native components such as FlatList and Navigation Stack. It includes a list of exercises and a timer or a rep counter based on the exercise of choice. The user can reset, pause, increase increment, and return to menu.

Quiz App

For this app, I created a 3-question quiz using React Native that supports three types of answers. When answered all correctly, the summary shows all green answers. If answered any incorrectly, the answer will show as red.

Webflow Website Redesign

I replicated an existing Webflow website using its template to further my understanding of the platform. Through this process, I gained a solid understanding of the main principles involved for website creation on Webflow.

Madden 25 Game Cover Redesign

I redesigned the cover of the game Madden 25 featuring Christian McCaffrey using Adobe Photoshop.

"Fyre" Energy Drink Concept

I created a design concept for a new energy drink brand with three flavors in Adobe Photoshop. "Fyre" is a modern take on energy drinks with a bold identity. Designed to attract a crowd who moves with purpose, the brand embodies momentum and "fyre power".

National Geographic Cover

I created my own cover for the National Geographic using my own photo in Adobe Photoshop.

Stephen Curry Olympics Graphic
