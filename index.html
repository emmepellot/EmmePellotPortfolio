Projects

To Do App

This app creates a "To Do" list and allows the user to check what they have completed, along with add new tasks to the list. I had some errors when passing each item to the renderItem component because I forgot to initialize with a useState. But I discovered my error and created a const that could handle an array of tasks.

Login and Registration App

This app uses React Native elements including the Stack Navigator to implement a Log in and Registration screen for the user to access the To Do App. I had the most difficulty validating each field in the registration screen, but I used regex and blur event handlers to implement it.

Exercise App

This is an exercise app I created using React Native components such as FlatList and Navigation Stack. I struggled the most with the reset button in each exercise, but I altered my functions to correctly clear the interval, stopping the timer, and set a variable to false, to then create a state variable that stores and updates the timer.

Quiz App

For this app, I created a 3 question quiz using React Native that supports three types of answers. I had difficulties highlighting the correct answers after the quiz was submitted, but I fixed this error in the Summary component, making the correct answer turn green.
